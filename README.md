# SortIt — Organize Your Downloads Automatically

SortIt automatically organizes your downloads into clean, structured folders — with zero effort.  
No more digging through a messy downloads folder, losing files, or wasting time searching. SortIt keeps everything tidy and easy to find.

---

## Why Install SortIt?

Your downloads folder is one of the most cluttered places on your computer. Every file from every website lands in a single chaotic pile. Files get buried, important documents disappear, and manual organization becomes a constant chore.

SortIt fixes that instantly and automatically.

---

## What SortIt Does

- Automatically sorts downloads into predefined categories such as **Documents**, **Images**, **Videos**, **Audio**, **Archives**, and more  
- Creates a dedicated **“SortIt – Organize Your Downloads”** folder for a clean and structured layout  
- Uses built-in logic to detect each file type and place it into the correct folder  
- Displays visual statistics so you can understand your download patterns  
- Provides quick-access category folders for instant navigation  
- Improves productivity by keeping your downloads organized at all times  

---

## Key Features

- Automatic sorting the moment a file downloads

## How It Works

1. Install SortIt  
2. The extension detects each file type and automatically sorts it into the right category  
3. Access category folders anytime from the popup  
4. Enjoy a clean, organized downloads folder — every day  

---

## Privacy & Security

SortIt does **not** collect or transmit personal data.  
All sorting happens locally on your device. Your files never leave your system.
